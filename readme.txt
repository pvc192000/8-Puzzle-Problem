8 Puzzle Problem Solver

To compile and run:

  1. Copy all the contents in a directory
  2. Run the makefile (if not using makefile make sure to have -std=c++17 flag while compiling)
      $ make
  3. The executable is stored as 'puzzle'. Run the executable
      $ ./puzzle
  4. Before running, make sure the file name matches with the text file names for running samples.
      The folder has two test text files: 1.txt and 2.txt. By default 8_puzzle_problem opens 1.txt
